How to run

```
1. Install dulu python 3
2. Install dulu pip3
3. pip install -r "requirement.txt"
4. pip install pillow
5. pip install pydub
6. pip install ffmpeg libavcodec
7. pip install ffmpy
8. export FLASK_APP=data-compress.py
9. export FLASK_ENV=development
10. flask run
11. buka localhost
```
